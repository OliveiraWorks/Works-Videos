## 1. Flappy Bird AI

Rede neural que aprende a jogar **Flappy Bird** sozinha, melhorando a cada geração de treinamento.

**Descrição:**  
Este projeto implementa uma **Rede Neural Artificial** que controla o pássaro no jogo Flappy Bird.  
Em vez do jogador apertar a tecla para pular, a IA decide se deve ou não pular com base nas informações do jogo (posição do pássaro, distância até o próximo cano, altura da abertura, etc.).

O treinamento é feito usando um algoritmo de aprendizado (como algoritmos genéticos / aprendizado por reforço), permitindo que o agente vá ficando cada vez melhor e sobreviva por mais tempo no jogo.

**Objetivos do projeto:**
- Praticar conceitos de **IA aplicada a jogos**  
- Entender melhor como funciona **treinamento de redes neurais**  
- Trabalhar com **simulação, lógica de jogo e tomada de decisão automática**

**Tecnologias utilizadas:**
- Linguagem: **Python**  
- Bibliotecas:**Pygame**

## 2. Análise de Imagens em Tempo Real

Sistema de **visão computacional** que analisa imagens da câmera em tempo real usando Inteligência Artificial.

**Descrição:**  
Este projeto captura frames da câmera e faz o processamento de cada imagem em tempo real, aplicando algoritmos de visão computacional e/ou modelos de IA para identificar objetos, padrões ou características específicas na cena.

A ideia é simular uma pipeline de visão computacional usada em aplicações reais, como monitoramento, segurança, automação e reconhecimento de objetos.

**Funcionalidades principais:**
- Captura de vídeo em tempo real  
- Processamento frame a frame  
- Aplicação de filtros, detecção ou classificação em tempo real  
- Exibição do resultado diretamente na tela

**Objetivos do projeto:**
- Explorar **OpenCV** e técnicas de **processamento de imagens**  
- Testar o uso de IA com baixa latência (resultado quase instantâneo)  
- Criar uma base para projetos mais avançados de visão computacional

**Tecnologias utilizadas:**
- Linguagem: **Python**  
- Biblioteca principal: **OpenCV**

## 3. GreenScan – Análise de Resíduos Sólidos por Imagem

Site que utiliza **IA e visão computacional** para auxiliar na identificação de resíduos sólidos a partir de imagens.

**Descrição:**  
A GreenScan é uma plataforma web onde o usuário envia uma foto de um resíduo (por exemplo: plástico, metal, papel, vidro) e o sistema realiza uma análise da imagem usando um modelo de IA para ajudar na classificação  do tipo de material marcando em um mapa aonde o descarte incorreto foi realizado e fazendo uma denuncia.

A proposta é mostrar como **Inteligência Artificial** pode ser aplicada em problemas ligados à **sustentabilidade** e ao manejo correto de resíduos.

**Funcionalidades principais:**
- Upload de imagem pelo usuário  
- Envio da imagem para processamento pela IA  
- Retorno de uma análise/classificação do tipo de resíduo  
- Interface simples e direta para testes e demonstração

**Objetivos do projeto:**
- Combinar **IA + Web** em um único projeto  
- Explorar o uso de visão computacional em um contexto de **meio ambiente**  
- Criar uma interface amigável para demonstrar o modelo de IA

**Tecnologias utilizadas:**
- Front-end: **HTML, CSS, JavaScript**  
- Back-end: **(Python/Flask)
- IA / Visão Computacional:(Modelo treinado por mim mesmo)
